🌍 AI Language Translator

An AI-powered web application that translates text between multiple languages instantly.
This project is developed using **Python Flask** and simple web technologies to provide a user-friendly language translation interface.

 🚀 Project Overview

The **AI Language Translator** allows users to enter text in one language and instantly translate it into another language.
It uses language processing libraries and a web interface to make translation simple and accessible.

This project was developed as part of the **CodeAlpha Internship Program**.

 ✨ Features

✔️ Translate text into multiple languages
✔️ Simple and clean user interface
✔️ Fast translation response
✔️ Easy to use web application
✔️ Built with Python and Flask

 🛠️ Technologies Used

 ⚙️ Installation & Setup

Follow these steps to run the project locally:

 1️⃣ Clone the Repository

git clone https://github.com/ajaygoud16/CodeAlpha_AI_Language_Translator.git

 2️⃣ Navigate to the Project Folder

cd CodeAlpha_AI_Language_Translator

 3️⃣ Install Required Libraries

pip install -r requirements.txt

4️⃣ Run the Application

python app.py

 5️⃣ Open in Browser

 http://127.0.0.1:5000
